Custome Case Application

Over 500,00 custome cases created.

Case-Mate has always been a leader in the custom case business, launching one of the first applications enabling users to upload images that can be printed onto a phone case.

Our assignment was to take the first iteration of this application and expand its feature set, improve the experience, create an updated image, and implement the solution using HTML5 to ensure it can be used across all devices., application interface that highligh.

We concepted and created a digital canvas so that the tools enabling you to manipulate the image are at your fingertips. A most challenging component of the project was to create an interface that was easy to use on a touch screen but didn’t seem cumbersome when using a mouse.
